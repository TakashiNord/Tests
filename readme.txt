
1. Требования к сборке ПО и работы.

Операционная система:
  Windows 7 SP1 (x86 и x64)
.NET:
  Microsoft .NET Framework 4 и выше
Runtime
  Visual C++ 2008 SP1 Runtime
Среда разработки:
  SharpDevelop 5 (http://www.icsharpcode.net/OpenSource/SD/Download/ )




2. Краткое описание работы.

  2.1 запустите приложение
  2.2 Выберите режим работы банкомата
      кнопка 'Выдача' - выдача денег
      кнопка 'Прием денег' - "принимает" в банкомат деньги.
  2.3 Режим Выдача.
   
      1. Нажимая кнопки '50\100\500\1000 рублей'
         банкомат выдает купюры и уменьшает общее количество денег у него.
      2. Если купюры данного номинала иссякнут - будет выдано сообщение.
      3. Нажимая кнопку 'Другая сумма'
         окно ввода Суммы становиться активным.
         В данное окно вы можете ввести необходимую сумму для выдачи.
         Также можете указать, типы купюр для выдачи суммы.
         Далее, нажимая кнопку 'Принять'
         - программа "выдает требуемую сумму" и уменьшает общее количество денег
           в Банке.

  2.4 Режим 'Прием денег'.

      1. Нажимая кнопки '50\100\500\1000 рублей'
         банкомат принимает купюры, увеличивая общее количество денег у себя.
      2. В случае, переполнения "кубышки", он откажется принимать купюру.
