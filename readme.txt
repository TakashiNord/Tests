
1. Требования к сборке ПО и работы.

Операционная система:
  Windows 7 SP1 (x86 и x64)
.NET:
  Microsoft .NET Framework 4 и выше
Runtime
  Visual C++ 2008 SP1 Runtime
Среда разработки:
  SharpDevelop 5 (http://www.icsharpcode.net/OpenSource/SD/Download/ )




2. Краткое описание работы.

  2.1 запустите приложение
  2.2 Перенесите файлы на область таблицы
      или добавьте их через кнопку 'Добавить Файл в список'
  2.3 Если необходимо удалить слова менее N символов. Задайте это число.
  2.4 Выделите необходимые знаки пунктуации, которые необходимо удалить.
  2.5 Запустите обработку файлов нажав кнопку 'Запуск обработки файлов'
  2.6 В процессе работы программы создается новый файл ( в этой же директории, что и 
     рабочий файл). В качестве имени используется имя рабочего файла + время создания.
